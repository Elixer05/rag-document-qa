# psychic-journey
Retrieval-Augmented Generation (RAG) with Chapter-Level Filtering
A Retrieval-Augmented Generation (RAG) system that performs semantic search over multi-chapter documents with metadata-based chapter filtering.

Features

Document chunking (~800-token segments) with chapter metadata

Dense vector embedding generation using transformer models

Chapter-specific filtering before similarity search

Cosine similarity ranking for semantic retrieval

Context-grounded answer generation using an LLM

Technologies Used

Python 3.x

NumPy

scikit-learn (cosine similarity)

Transformer-based embedding model

LLM integration(Ollama)
